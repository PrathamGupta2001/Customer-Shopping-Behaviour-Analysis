# 📌 Project Overview

The objective of this project is to simulate a **corporate-grade end-to-end data analytics workflow**, demonstrating how raw data can be transformed into meaningful business intelligence and actionable insights.

1. Data Preparation, Modeling & Exploratory Data Analysis (Python)

* Cleaned and transformed raw customer data.
* Handled missing values and data inconsistencies.
* Performed Exploratory Data Analysis (EDA) to uncover patterns and trends.

2. Data Analysis (SQL)

* Simulated business scenarios and transactions.
* Executed SQL queries to answer key business questions.
* Analyzed customer segments, loyalty, and purchasing behavior.

3. Visualization & Insights (Power BI)

* Designed an interactive dashboard for stakeholders.
* Visualized customer trends and performance metrics.
* Enabled data-driven decision-making through meaningful insights.

4. Report & Presentation

* Created a detailed project report summarizing findings.
* Provided business recommendations based on analysis.
* Developed a presentation deck using Gamma AI to communicate insights effectively.

🛠️ How to Use This Project

1. Clone the Repository


git clone https://github.com/PrathamGupta2001/Customer-Shopping-Behaviour-Analysis.git


2. Open Customer_Shopping_Behavior_Analysis.ipynb

This notebook contains:

* Data import and loading
* Data exploration
* Data cleaning and preprocessing
* Connection setup for SQL database

3. Load Data into SQL Database

* Create a database in MySQL, PostgreSQL, or SQL Server.
* Run Python scripts to load the cleaned dataset into the database.

4. Open customer_behavior_sql_queries.sql

* Execute SQL queries to answer business questions.
* Extract insights related to customer behavior and purchasing trends.

5. Connect SQL Database to Power BI

* Import the SQL database into Power BI.
* Open customer_behavior_dashboard.pbix.
* Create interactive visualizations and dashboards.

6. Create Report and Presentation

* Prepare a project report summarizing findings and recommendations.
* Build a presentation deck using Gamma AI for stakeholder communication.
